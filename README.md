# AI Document Chatbot (RAG)

This is a simple document question-answering chatbot built using LangChain, HuggingFace, ChromaDB, and Streamlit.

The chatbot reads PDF and text documents, retrieves relevant content, and generates answers using a language model.

---

## Features

* Supports PDF and TXT files
* Semantic search using embeddings
* Chroma vector database
* T5-Small for answer generation
* Streamlit user interface

---

## How to run

Install requirements:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```
